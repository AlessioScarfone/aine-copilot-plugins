design systems

The design plugin turns any DESIGN.md into a production-ready HTML document. Tokens map directly to CSS — colors, type scale, shadows, radius — with no templates overriding your brand and no external dependencies.

typeset-freemaximum fidelity

Builds from scratch. Every CSS value is derived from DESIGN.md tokens.

  • Dark-mode and screen-first design systems
  • Brand-specific typefaces and component patterns
  • All CSS inline — single self-contained file
  • Best when visual precision beats layout predictability

typeset-printtemplate-backed

Maps tokens onto battle-tested print templates. Page breaks and WeasyPrint quirks handled.

  • Resumes, equity reports, changelogs
  • WeasyPrint-safe — no rendering surprises
  • Reliable A4 layout across all document types
  • Best when print predictability matters most
